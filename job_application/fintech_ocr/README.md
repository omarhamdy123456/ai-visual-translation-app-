# Fintech OCR Job Application Pack

Files in this folder support proposals for **OCR Specialist / Backend Developer (Fintech API)** roles.

## How to use

| File | Use for |
|------|---------|
| **Cover_Letter.md** | Full cover letter — customize `[Your Name]`, contact info, GitHub URL |
| **Proposal_Bullet_Points.txt** | Paste into "Additional details" or short proposal fields |
| **Project_Summary_One_Pager.md** | Attach as PDF or link in proposal |
| **Requirements_Mapping.md** | Interview prep; maps job reqs to your project |
| **Beta_Timeline_Milestones.md** | Fixed-price / milestone quote |

## Before you submit

1. Replace all `[Your Name]`, `[Email]`, `[GitHub]` placeholders.
2. Add a **public GitHub link** if the repo is private (or export a sanitized demo).
3. Export **Cover_Letter.md** or **Project_Summary_One_Pager.md** to PDF if the platform accepts attachments.
4. Optional: screenshot Annotated OCR tab + one chart from `experimental_results/`.

## Evidence files in main project (cite in proposal)

- `paddle_ocr_worker.py` — OCR worker
- `eval_experimental_results.py` — CER/WER benchmarks
- `experimental_results/ocr_results.json` — Arabic metrics
- `experimental_results/funsd_fullpage_test.json` — English full-page metrics
